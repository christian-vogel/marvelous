## Installation

### Clone Repository

Change to your Visual Studio Code extensions directory:

```bash
    # Windows
    C:\ cd %USERPROFILE%\.vscode\extensions

    # Linux & macOS
    $ cd ~/.vscode/extensions/
```

Clone repository:

```bash
$ git clone https://github.com/christian-vogel/marvelous.git
```
